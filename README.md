# beef-in-termux
Install Beef in Termux without any problems or errors with help and support if you find any problem

# Intalition
Packages needed
```bash
apt install -y git
```
Install beef
```bash
git clone --depth=1 https://github.com/ahmed-alnassif/beef-in-termux
cd beef-in-termux
chmod +X install.sh
./install.sh
```
